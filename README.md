# Simple ATM Machine (Java)

This is a simple console-based ATM application built using Java.  
It demonstrates the use of Object-Oriented Programming (OOP), methods, and switch-case.

## Features
- PIN Login
- Check Balance
- Deposit Money
- Withdraw Money
- Exit Option

## Technologies Used
- Java
- Scanner Class
- OOP Principles

## How to Run
1. Clone this repository:
2. Open the project in VS Code or any IDE.
3. Run the `ATMSystem.java` file.
